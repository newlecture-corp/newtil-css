# border-radius

[border-radius](https://developer.mozilla.org/en-US/docs/Web/CSS/border-radius) 속성에 대한 유틸리티입니다.

요소의 테두리의 둥글기를 설정합니다.

기본 테두리 둥글기 `var(--border-radius)` 설정과 관련된 내용은 [여기](/guide/css-variable-list.html#border-radius)를 참고해주세요.
