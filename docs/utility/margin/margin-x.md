# margin-x

margin-x 속성에 대한 유틸리티입니다.

요소 왼쪽과 오른쪽의 여백 영역([margin area](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_box_model/Introduction_to_the_CSS_box_model#margin_area))의 크기를 설정합니다.
