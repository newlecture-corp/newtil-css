# text-transform

[text-transform](https://developer.mozilla.org/en-US/docs/Web/CSS/text-transform) 속성에 대한 유틸리티입니다.

요소의 텍스트를 대문자화하는 방식을 지정합니다. 이를 사용하여 텍스트를 모두 대문자 또는 모두 소문자로 표시하거나 각 단어를 대문자로 표시할 수 있습니다.
